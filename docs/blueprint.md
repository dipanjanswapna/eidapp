# **App Name**: Eid-E-Salami

## Core Features:

- Personalized Salami Profile Setup: Users create a profile by entering their name, payment (bKash/Nagad) numbers, and choosing a visual card theme to receive digital 'Salami'.
- AI-Generated Funny Salami Message Tool: An AI tool that helps users generate creative and humorous 'salami demand' messages or slogans to personalize their cards.
- Unique Salami Link & Shareable Image Generation: Automatically generates a unique public URL for the user's Salami page and a custom, visually appealing image card for social sharing, featuring their details, AI-generated message, and a QR code.
- Public Salami Collection Page: A dedicated public page where benefactors can view the recipient's Salami card, see their funny message, and interact with payment options.
- Payment Deep Linking Integration: Features buttons that leverage deep linking to open bKash/Nagad mobile applications directly with the recipient's phone number pre-filled for effortless digital payment.
- Salami Wish Submission: Allows benefactors to submit a personalized text 'Salami Wish' message, which will be displayed on the recipient's public Salami page, adding a personal touch.
- Multi-Language UI Toggle: Provides a user interface toggle to switch between Bangla and English language for all app text, supporting a wider audience.

## Style Guidelines:

- Primary color: A vibrant, celebratory green (#2C9915) reflecting growth and festivity.
- Background color: A very light, desaturated green-ish white (#ECF5EB) providing a clean and soft canvas for the festive elements.
- Accent color: A bright, festive gold (#F7D40F) to highlight key actions and celebratory elements, harmonizing with the green.
- Headline and body font: 'Roboto' (sans-serif) for English content and general UI, paired with 'Hind Siliguri' (sans-serif) for Bengali content. Note: currently only Google Fonts are supported.
- Utilize modern, clean line icons that convey celebration, money transactions, sharing, and connectivity, aligning with the festive and digital theme.
- Adopt a mobile-first responsive design approach, ensuring optimal viewing and interaction across all device sizes for accessibility.
- Incorporate subtle, festive animations such as falling confetti or gently animated lanterns to enhance the celebratory and engaging user experience.
